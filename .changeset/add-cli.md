"@datoteca/cli": minor
---

Adds `@datoteca/cli`, a new package exposing the existing `@datoteca/cl` generators (`rut`, `person`, `address`, `phone`, `money`, `company`) as terminal subcommands (`npx @datoteca/cli <command>`), with `json`/`csv`/`ndjson` output and the same seeded determinism guarantee.
